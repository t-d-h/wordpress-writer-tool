# Plan: UI Components - Token Usage Display

**Phase:** 01 - Token Usage Display
**Wave:** 3
**Status:** Not started

**type:** implementation

## Objective

Create TokenUsageCard component and integrate it into ProjectDetail general tab to display token usage breakdown.

## Requirements

- TOKEN-01: User can view token usage breakdown in Project general tab above statistics section
- TOKEN-02: System displays token usage breakdown by post type (research, outline, content, thumbnail)
- TOKEN-03: System shows total input tokens and total output tokens across all post types
- TOKEN-07: Token usage display is always visible when viewing project details
- UX-01: Token usage display is visually distinct from existing statistics

## must_haves

### truths
- User can view token usage breakdown in Project general tab above statistics section
- System displays token usage breakdown by post type (research, outline, content, thumbnail)
- System shows total input tokens and total output tokens across all post types
- System includes deleted posts in token usage calculations
- Token usage display loads within 1 second for projects with <100 posts

### artifacts
- `frontend/src/components/Projects/TokenUsageCard.jsx` - Token usage display component
- `frontend/src/components/Projects/ProjectDetail.jsx` - Integrated component in general tab

### key_links
- TokenUsageCard component → ProjectDetail general tab
- Token usage state → TokenUsageCard prop
- Backend API → Token usage data source
