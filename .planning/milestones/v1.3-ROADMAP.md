## Technical Details

### HTML Cleaning Algorithm (5 stages)
1. Remove markdown code blocks (```...```)
2. Remove inline code backticks
3. Parse HTML with BeautifulSoup
4. Remove disallowed tags (script, div, style, iframe, span, etc.)
5. Preserve only safe attributes (href, src, alt, title)

### Allowed HTML Tags
h1, h2, h3, h4, h5, h6, p, strong, em, ul, ol, li, a, img

### Safe HTML Attributes
href, src, alt, title
