## Implementation Notes

### Design Decisions
- Radio buttons (not checkbox) for mutually exclusive choice
- Vietnamese as default language (per requirements)
- Color coding: green for Vietnamese, blue for English
- Language badge displayed in both list and detail views

### Technical Approach
- Use React state for form language selection
- Use localStorage for persistence across page refreshes
- Extract language from post data for badge display
- Default to "English" badge for posts without language field (backward compatibility)
