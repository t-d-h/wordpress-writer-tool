# Plan: Add Static News Tab

## Overview
Add a new News tab to the application that displays static HTML content without any backend integration.

## Implementation Steps

### 1. Create News Component
**File**: `frontend/src/components/News.jsx`

Create a new React component that displays static news content. The component should:
- Use the same styling patterns as other pages (page-header, page-title, page-description)
- Display static HTML content (news items, updates, announcements)
- Be fully self-contained with no API calls
- Follow the existing component structure

**Suggested content structure**:
- Page header with title "News" and description
- News items displayed as cards or list items
- Each news item should have:
  - Title
  - Date
  - Content/description
  - Optional tags or categories

### 2. Add News Route
**File**: `frontend/src/App.jsx`

Add the News route to the routing configuration:
- Import the News component
- Add route: `<Route path="news" element={<News />} />`
- Place it in the protected routes section (after line 30, before line 31)

### 3. Add News Navigation Link
**File**: `frontend/src/components/Sidebar.jsx`

Add a News link to the sidebar navigation:
- Add a new nav item in the "Menu" section (after line 78, before the Settings section)
- Use an appropriate icon (e.g., `HiOutlineNewspaper` or `HiOutlineSparkles`)
- Make it a NavLink to `/news`
- Follow the existing pattern for nav items

## Files to Modify
1. **Create**: `frontend/src/components/News.jsx` - New component with static content
2. **Modify**: `frontend/src/App.jsx` - Add News route
3. **Modify**: `frontend/src/components/Sidebar.jsx` - Add News navigation link

## Design Considerations
- Use existing CSS classes for consistency (page-header, page-title, page-description, etc.)
- News items can be styled as cards similar to project cards
- Include sample news content that can be easily edited later
- No state management needed (static content only)
- No API calls or data fetching required

## Sample News Content Structure
```jsx
<div className="page-header">
  <h1 className="page-title">News</h1>
  <p className="page-description">Latest updates and announcements</p>
</div>

<div className="news-list">
  <div className="news-card">
    <div className="news-date">April 19, 2026</div>
    <h3 className="news-title">New Feature: AI Content Generation</h3>
    <p className="news-content">We've added powerful AI content generation...</p>
  </div>
  <!-- More news items -->
</div>
```

## Testing Strategy
1. Verify the News link appears in the sidebar
2. Click the News link and verify it navigates to `/news`
3. Verify the News page displays correctly with static content
4. Verify the styling matches other pages in the application
5. Verify no API calls are made (check browser console/network tab)

## Notes
- This is a frontend-only feature with no backend dependencies
- Content is static and can be updated by editing the News.jsx file
- No database or API integration required
