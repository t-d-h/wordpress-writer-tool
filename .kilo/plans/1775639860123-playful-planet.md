# Plan: Simplify Token Tracking

## Overview
Remove input/output token separation and simplify to track only total tokens per stage. This reduces complexity and removes unnecessary code while maintaining the core functionality of tracking token usage.

## Changes Required

### 1. Backend Changes

#### 1.1 Update TokenUsage Model (`backend/app/models/post.py`)
- Remove `input_tokens` and `output_tokens` fields
- Keep existing per-stage fields (research, outline, content, thumbnail, section_images, total)

```python
class TokenUsage(BaseModel):
    research: int = 0
    outline: int = 0
    content: int = 0
    thumbnail: int = 0
    section_images: int = 0
    total: int = 0
    # REMOVED: input_tokens, output_tokens
```

#### 1.2 Simplify AI Service Functions (`backend/app/services/ai_service.py`)
- Modify `_call_openai`, `_call_gemini`, `_call_anthropic` to return `(text, total_tokens)` tuple
- Update all service functions to return only total tokens
- Remove input/output token extraction logic

**Before:**
```python
async def _call_openai(...) -> tuple[str, int, int]:
    input_tokens = response.usage.input_tokens if response.usage else 0
    output_tokens = response.usage.output_tokens if response.usage else 0
    return response.choices[0].message.content, input_tokens, output_tokens
```

**After:**
```python
async def _call_openai(...) -> tuple[str, int]:
    total_tokens = response.usage.total_tokens if response.usage else 0
    return response.choices[0].message.content, total_tokens
```

#### 1.3 Update Worker Tasks (`worker/app/workers/tasks.py`)
- Update `run_research`, `run_outline`, `run_content` to track only total tokens
- Remove input/output token storage and logging
- Simplify token calculation

**Before:**
```python
research_data, input_tokens, output_tokens = await ai_service.research_topic(...)
total_tokens = input_tokens + output_tokens
await posts_col.update_one(
    {"_id": ObjectId(post_id)},
    {
        "$set": {
            "token_usage.research": total_tokens,
            "token_usage.input_tokens": input_tokens,
            "token_usage.output_tokens": output_tokens,
        }
    },
)
```

**After:**
```python
research_data, total_tokens = await ai_service.research_topic(...)
await posts_col.update_one(
    {"_id": ObjectId(post_id)},
    {
        "$set": {
            "token_usage.research": total_tokens,
        }
    },
)
```

#### 1.4 Update Worker AI Service (`worker/app/services/ai_service.py`)
- Apply same changes as backend AI service
- Ensure consistency between backend and worker
